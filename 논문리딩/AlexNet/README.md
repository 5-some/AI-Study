# [Paper Summary] AlexNet: Deep Convolutional Neural Networks

> **핵심 요약:** 기존의 수동적인 특징 추출 방식(Hand-crafted feature)의 한계를 극복하고, 대규모 데이터와 GPU 파워를 결합해 딥러닝의 가능성을 증명한 기념비적인 논문입니다.

---
 
## 1. 개요 및 배경 (Background)

###  왜 이 논문이 나왔는가?
과거에는 이미지 인식에 "사람이 직접 특징을 뽑아주는" 방식(SIFT, FV 등)을 썼습니다. 예를 들어 "귀가 뾰족하면 고양이"처럼 규칙을 사람이 직접 설계했죠. 하지만 현실 세계의 이미지는 변수가 너무 많아 사람이 만든 규칙만으로는 한계에 부딪혔습니다.

###  기존 방식의 문제점
1. **데이터셋의 한계:** 학습 데이터가 수만 장 수준으로 적어 복잡한 패턴 학습이 어려웠습니다.
2. **수동적 특징 설계:** 사람이 직접 특징을 정의하므로, 기계가 스스로 복잡한 추상적 패턴을 찾아내지 못했습니다.
3. **연산 능력 부족:** CNN은 이론적으로 훌륭했지만, 당시 컴퓨터 성능으로는 대규모 신경망을 학습시키기에 너무 느렸습니다.

###  AlexNet의 주요 개선점 (Solution)
* **대규모 병렬 학습:** GPU 2개를 연결하여 연산 한계를 돌파했습니다.
* **학습 속도 혁신:** ReLU 활성화 함수를 도입해 기존 Sigmoid 대비 학습 속도를 6배 이상 높였습니다.
* **강력한 규제(Regularization):** Dropout과 데이터 증강을 통해 과적합(Overfitting) 문제를 정면으로 해결했습니다.
* **깊이의 가치 증명:** "층이 깊어질수록 성능이 좋아진다"는 것을 실험으로 입증했습니다.

## 2. 핵심 구조 및 기술 상세

### ① Convolutional Layer (합성곱 레이어)
* **비유:** 돋보기로 사진을 구석구석 훑으며 패턴을 찾는 과정
* **내용:** 이미지 위를 필터가 슬라이딩하며 엣지, 질감 등의 특징을 추출합니다. AlexNet은 총 5개의 Conv 레이어를 쌓아 깊은 특징을 추출했습니다.

### ② ReLU Nonlinearity (활성화 함수) ⭐ 핵심
* **문제 해결:** 기존 Tanh나 Sigmoid는 값이 커질수록 기울기가 사라지는 '기울기 소실(Vanishing Gradient)' 문제가 있었습니다.
* **해결:** $f(x) = \max(0, x)$ 수식을 사용해 양수 구간에서 기울기를 유지함으로써 **6배 빠른 학습 속도**를 달성했습니다.

### ③ Local Response Normalization (LRN) ⭐ 핵심
* **비유:** 학교에서 한 명의 천재보다 여러 명의 우등생이 조화롭게 경쟁하도록 유도하는 것
* **내용:** 강하게 활성화된 뉴런이 주변 뉴런을 억제하는 '측면 억제(Lateral Inhibition)'를 모방했습니다. 최근엔 잘 안 쓰이지만, 당시에는 일반화 성능을 높이는 데 큰 역할을 했습니다.

### ④ Overlapping Max Pooling (중첩 풀링) ⭐ 핵심
* **비유:** 사진을 축소하되, 영역을 겹쳐서 축소하여 정보 손실을 최소화하는 것
* **내용:** 보편적인 풀링과 달리 윈도우를 겹치게 설정(z=3, s=2)하여 과적합을 줄이고 에러율을 0.4%p 가량 낮췄습니다.

### ⑤ Data Augmentation (데이터 증강) ⭐ 과적합 방지 1
* **방법:** 256x256 이미지를 무작위로 자르고(Crop), 좌우 반전시키며, RGB 채널의 강도를 변형했습니다.
* **효과:** 한정된 데이터를 수백 배로 늘려 모델이 이미지의 본질적인 특징을 학습하게 유도했습니다.

### ⑥ Dropout (드롭아웃) ⭐ 과적합 방지 2
* **비유:** 훈련 때 주전 선수를 일부러 빼서 후보 선수들의 실력을 골고루 키우는 팀 전략
* **내용:** 학습 시 뉴런의 50%를 무작위로 꺼서 특정 뉴런에만 의존하는 현상을 막았습니다. 연산량은 늘어나지만 과적합 방지에 매우 효과적이었습니다.

### ⑦ Softmax (최종 출력)
* **내용:** 모델이 찾은 특징들을 종합해 1,000개의 클래스별 확률값(합계 1.0)으로 변환합니다. 가장 높은 확률을 가진 클래스가 최종 예측값이 됩니다.

## 3. 결과 및 지표 분석 (Top-1 & Top-5 Error)

| 지표 | AlexNet 성적 | 비고 |
| :--- | :---: | :--- |
| **Top-1 Error** | 37.5% | 1순위 예측이 틀린 비율 |
| **Top-5 Error** | **15.3%** | 상위 5개 중 정답이 없을 비율 (우승 지표) |

###  왜 Top-5 Error가 중요한가요?
1. **분류의 모호성 인정:** '시베리안 허스키'와 '에스키모 개'처럼 사람이 봐도 헷갈리는 유사 클래스가 많습니다. 상위 5개 안에 정답이 있다면 모델이 사물의 특징은 제대로 파악했다고 보는 것입니다.
2. **모델의 이해력 측정:** 단순히 하나를 찍어 맞추는 운보다, 모델이 정답 후보군을 얼마나 합리적으로 압축했는지 평가하는 더 정교한 척도입니다.
