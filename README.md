# AI-Study

AI 공부를 기록하는 공간

## 무엇을 기록하나요?
논문 분석과 코드 문제풀이를 기록하며 실력을 향상시킵니다.

## 📖 스터디 목표
* 핵심 논문들을 분석하며 코드리딩 능력을 키우기  
* 파이썬과 라이브러리 문법을 익히며 결과물을 GIthub에 업로드하기  
* 논문 기반의 코드를 실현하여, 결과물을 Git & HuggingFace 에 업로드  
* HuggingFace에서 모델을 불러와 실행 및 Gradio 데모를 제작  


## 🔍 운영구조
### 매일 — Python 문제풀이 30분 및  업로드
* 파이썬 기초 300제 or 민코딩 활용 자기 수준에 맞는 문제 풀기  
* Colab 노트북으로(코드에 풀이 과정 주석 달아서) 풀고 GIthub 업로드 후 단톡방에 링크공유  
* 짧고 간단하게 — 하루 30분 내외  
* 목적: Python 기초 문법 익숙해지기 + Colab 및 Github 사용법 익히기  

### 화·수·목 — 개인 자율 학습 (권장 2시간)
* 해당 주 논문 읽기 (회독 전략 추천, 아래 참고)  
* 논문 기반 코드 구현 후 GitHub 및 HuggingFace 업로드(직접 다 안짜도 됨)   
* 요일은 꼭 화 수 목 아니어도 되고 조율 가능  
* 단, 금요일까지 Git + Hugging 업로드는 엄수(주말 스터디 전까지)  

### 금요일 — 주간 점검 (30분)
* 수업 후 짧게 모여 이번 주 공부 현황 공유  
* 주말 스터디 전, 어려운 부분 1차 공유  
* 주말 발표 준비 잘 되고 있는지 확인  

### 토요일 or 일요일 — 메인 스터디 세션 (2~3시간)
매주 2인 1조 1팀이 발표합니다. 6명 3조가 돌아가며 진행합니다.
